# Goal

Rewrite the `remote` command of the `gt` tool in Rust.

# Instructions

1. Setup the dev environment for Rust with the standard linter and formatter.
2. Translate the `remote` command that is currently implemented in `gt-remote.sh` to a corresponding Rust implementation.
3. Create test for the Rust implementation that covers the newly created code.
4. Compile and verify that the resulting code is fully runnable.

# Requirements

- Only translate the remote command, not the other commands yet.
- Write it in a way so that the other commands can be added later.
